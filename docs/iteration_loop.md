# Post‑launch Iteration Loop

## Еженедельный review
- TTS, completion, replay, share.
- Топ‑dropoffs и узкие места.
- Решения только про опыт (смех/темп).

## Обновление контента
- Добавление/замена сцен и ассетов.
- Соблюдение guardrails: без брендов/персоналий/политики/религии.

## A/B тесты (без текста)
- Темп раунда.
- Длительность таймера.
- Порядок сцен.
- Визуальный стиль наград.

## Анти‑red‑ocean фильтр
- Не добавлять чтение, викторины, сложные правила.
- Не оценивать «по вкусу».
