# License Policy

## Принимаемые лицензии
- CC0
- Public Domain (PD)
- Pixabay License

## Процесс валидации
- Проверка страницы файла вручную.
- Фиксация типа лицензии и даты проверки.
- Запись в `assets/manifest.json` (license, source_url, evidence_file, license_checked_at).

## Аудит контента
- Авто‑blacklist IP/брендов/персоналий.
- Ручная выборочная проверка.
- Запрет политических/религиозных сюжетов.

## Хранение доказательств
- Снимок страницы/лицензии сохраняется в `licenses/`.
- Ссылка на источник хранится в манифесте ассетов.

## Политика обновления
- Каждый новый ассет требует записи лицензии.
- При повторном использовании — проверка и обновление даты.
