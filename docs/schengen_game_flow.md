# Schengen: сценарий (только визуал)

## Цель
- Уйти от чтения: только визуальные сигналы, крупные иконки, быстрые реакции.
- Ритуал смеха: общий темп, минимум шагов, максимум «угара».
- Kahoot‑like: общий экран (host) + телефоны игроков.

## Роли
- Host/экран: показывает раунд, таймеры, распределение, лидерборд, награды.
- Игроки: выбирают ответы/реагируют на телефонах (тап/свайп).
- Наблюдатели: смотрят общий экран, входят в игру со следующего раунда.

## Основной сценарий (host → lobby → round loop → results → leaderboard → reward)
1) Host создаёт комнату.
2) Lobby: игроки заходят через QR/PIN/линк. Аватары появляются, авто‑ready.
3) Countdown: визуальный круг/пульс до старта.
4) Round loop (10 типов сцен):
   - prompt (визуальный стимул)
   - ответы (4 визуальные зоны)
   - реакция (раскрытие/эффект)
   - распределение ответов (без чисел)
   - лидерборд (иконки/медали)
   - награда/ритуал
5) Результаты матча и share‑карточка.
6) Play‑again: один большой тап, новый круг.

## Офлайн‑сценарий (вечеринка в одной комнате)
- Экран «Создать вечеринку»: одна большая кнопка‑иконка, авто‑создание комнаты.
- Сразу показывается QR d0.do/XXX, крупный, с рамкой и анимацией.
- Игроки сканируют, попадают в join без текста, выбирают аватар одним тапом.
- Lobby: ряд аватаров, пульсация при входе, авто‑ready.
- Старт через countdown.

## Онлайн‑сценарий (вход в идущую игру)
- Экран «Войти в игру»: один CTA (скан QR/вставить PIN/быстрый вход).
- Если игра идёт: join‑in‑progress → визуальный экран ожидания.
- Вход в следующий раунд без текста, авто‑ready.

## QR‑интеграция
- Генерация комнаты: d0.do/XXX.
- Повторный показ QR одним тапом (иконка QR).
- Увеличение QR жестом/тапом.
- Fallback: PIN/линк (иконка клавиатуры/цепочки), без текста.

## Round loop (ASCII)
```
[Prompt] -> [4 ответа] -> [Реакция] -> [Распределение] -> [Лидерборд] -> [Награда] -> [Next]
     ^                                                                            |
     |----------------------------------------------------------------------------|
```

## 10 типов сцен и связь с циклом
- visual_provocation: стартовый визуальный триггер → 4 визуальных ответа.
- face_mimic: маска/мимика → 4 реакции.
- sound_pantomime: звуковой символ → 4 ассоциации.
- silhouette_guess: силуэт → 4 формы.
- telepath_sync: совпадение без «правильного» → общая реакция.
- icon_battle: две иконки против → выбор.
- absurd_sum: «X + Y» визуально → 4 абсурдных варианта.
- absurd_toast: визуальный «тост» → 4 настроения.
- drunk_reflex: быстрый сигнал → 4 реакции.
- game_story: короткая визуальная мини‑история → финальный выбор.

Все сцены проходят общий loop: prompt → ответы → реакция → распределение → лидерборд → награда.

## Пьяный быстрый старт
- 2–3 тапа: создать → скан → старт.
- Никаких правил/текстовых подсказок, только визуальные символы.

## Anti‑red‑ocean guardrails
- visual_provocation: только эмоции/абсурд, без брендов/персоналий.
- face_mimic: мимика без слов/цитат/политики.
- sound_pantomime: звуковые ассоциации без песен/бренд‑аудио.
- silhouette_guess: формы без логотипов/селебрити.
- telepath_sync: совпадения без «правильного ответа».
- icon_battle: иконки/символы без торговых марок.
- absurd_sum: абсурд без новостей/религии/политики.
- absurd_toast: тёплый «тост» визуально, без текста/брендов.
- drunk_reflex: реакция без токсичных/опасных сцен.
- game_story: мини‑истории без текста и без IP.

## Аудитория 21–55
- 21–30: быстрый темп, яркие эффекты.
- 31–40: баланс темпа и ясности.
- 41–55: мягкие эффекты, спокойные цвета, меньше перегруза.
- Во всех режимах: обучение через визуальные подсказки.
